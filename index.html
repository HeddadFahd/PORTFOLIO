Portail d'Actualités sur l'IA

Web

Description du projet

Un site web d'actualités moderne dédié à l'intelligence artificielle, développé avec HTML, Tailwind CSS et JavaScript. Ce portail présente les dernières avancées de l'IA dans différents secteurs comme la santé, l'automobile et la finance.

Le site comprend un système de filtrage par catégories, un carrousel d'images pour les actualités importantes, et une interface utilisateur intuitive avec un thème sombre élégant. L'accent a été mis sur l'expérience utilisateur et la performance.

Technologies utilisées

HTML5 CSS3 / Tailwind CSS JavaScript Responsive Design API REST

Caractéristiques principales

  • Interface utilisateur intuitive avec un thème sombre élégant
  • Carrousel d'images automatique pour les actualités importantes
  • Navigation par catégories (Santé, Automobile, Finance, etc.)
  • Design responsive pour une expérience optimale sur tous les appareils
  • Intégration d'API pour les données d'actualités en temps réel
